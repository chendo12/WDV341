Git Related Terms

  1. Version Control Software - Is the managing and tracking the changes done to code. The benifits of this is that multiple people can be working on one project at the same time and not interfere with other peoples code.
  2. Add - The add command updates a change that was made to the code, it tells git that you will like to add the change on the next commit.
  3. Commit - The commit command makes the updates and changes made and also includes any log messages that were added along side the changes
  4. Push - The push command is used to upload local Repository to a remote Repository. Pushing is how you upload your local Repository to your remote Repository.
  5. Pull- The pull command is the opposite of the push. The pull command pulls content from your remote Repository to your local Repository.
  6. Clone - The clone command does excatly what the name indicates, it clones a Repository into a newly created directory.
